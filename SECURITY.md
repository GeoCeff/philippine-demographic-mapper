# Security Policy

## Supported Versions

This project is in early prototype stage. Security fixes should target the latest commit on `master`.

## Reporting a Vulnerability

Please open a private security advisory on GitHub if available, or contact the repository owner directly.

Include:

- A short description of the issue.
- Steps to reproduce.
- Expected impact.
- Any suggested fix.

Please do not publicly disclose exploitable issues until there has been time to review and patch them.
