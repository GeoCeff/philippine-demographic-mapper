# Sample CSVs

Use these in the app's CSV importer:

- `region_values.csv`: national region-level values.
- `province_values.csv`: national province-level values.
- `barangay_values.csv`: Quezon City barangay-level values.
- `match_issues_demo.csv`: intentionally messy rows for testing duplicate, missing, wrong-level, outside-scope, unknown-code, and invalid-value review states.
